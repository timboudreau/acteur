com.mastfrog.acteur.output

This package contains classes for writing output from streams of various sorts scalably, emitting data in small chunks so that a request cannot pull very large amounts of data into memory.

For each case, an Acteur is provided which expects the stream-type (such as a Stream or InputStream) to have been made available in the request scope by a preceding Acteur, by calling next(theStream).

The stream handling acteur will take care of batching output until all of the data has been written (or the client has closed the connection) and will ensure the stream gets closed when the connection does, if not sooner.
